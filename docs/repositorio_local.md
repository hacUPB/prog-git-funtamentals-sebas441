Para crear un repositorio local con Git, primero debes tener una carpeta para tu proyecto.
Luego inicializas Git dentro de esa carpeta para que empiece a controlar los cambios.
Después agregas los archivos del proyecto al control de versiones y guardas el estado inicial
creando un primer registro de cambios (commit).
A partir de ahí, Git llevará el historial de todas las modificaciones que hagas.
